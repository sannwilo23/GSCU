# 全球亚文化俱乐部联盟宣言

自二十一世纪以来，这世上有许许多多的亚文化不断发展，又有许多新的亚文化诞生，2020 年 COVID-19 疫情以前，亚文化在绝大多数人看来只用于休闲。

然而，2020 年，疫情爆发并且持续了三年，在疫情期间和结束后，这世上有不计其数的年轻人找不到工作，就算找到了工作，收入也难以维持接下来的生活。

于是，亚文化对于这些年轻人，不再只是生活休闲的一部分，而是"避风港"。

此后，亚文化运动在全球各地爆发，到了 2030 年左右已成燎原之势，亚文化的发展达到了前所未有的繁荣。

然而，各国政府却认为这些年轻人在当鸵鸟，他们使出了各种招数试图遏制。

但接下来，亚文化运动进行得更激进、更隐蔽。现在这些年轻人当中的一小部分，正准备在未来几十年里干出一番大事业————一个围绕亚文化这个主题的联盟，即将来到这个世界。

而这个联盟的名字，就是"全球亚文化俱乐部联盟"（Global Subculture Clubs Union），简称 GSCU。

自此宣言签署开始，GSCU，将正式成立。

此后，GSCU 将为人类带来希望与曙光，保护和发展好每一个美好的亚文化，让世界更加多样化！

我们的代号即是我们的勋章，我们的据点即是人类想象力的灯塔。自本宣言签署之时起，我们将不为功利折腰，不向资本低头。世界的未来和理想，交由每一名研究员守护。

GSCU 永不灭亡！
