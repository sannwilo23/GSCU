## 📖 项目简介

这是一个设定在 **2040-2060 年** 的近未来科幻世界观项目，描绘了**全球亚文化俱乐部联盟（GSCU）**——一个由数千万"研究员"组成的去中心化亚文化保护组织的兴衰与未来。

本设定集采用 **"伪档案体"** 写作风格，以 2060 年联盟成立 20 周年（UTC 2060-04-08 0:00）为叙事时点，通过官方文档、技术规范、历史记录和成员留言等多重视角，构建一个既乌托邦又充满现实张力的未来社会图景。

### 核心设定亮点

- **去中心化治理**：灵感源自 Debian 社区，无总部、无公司、全球联邦制自治
- **精密编号系统**：GPC（GSCU Pointer Code）——从俱乐部代号到个人编号的完整层级体系
- **技术与社会冲突**：量子计算、脑机接口与复古树莓派并存的技术分化
- **真实感细节**：从 2.52 平方米的微型据点到 10 平方公里的街区，从树莓派服务器到 FeRAM 芯片升级

## 🎯 阅读指南

### 快速入门（15 分钟）

1. 先读 **[全球亚文化俱乐部联盟.md](./GSCU介绍及附件文档/全球亚文化俱乐部联盟.md)** 的"**联盟构成**"和"**联盟的发展**"章节
2. 了解 **[GSCU 联盟契约.md](./GSCU介绍及附件文档/GSCU联盟契约.md)** 的 5 条核心承诺
3. 浏览 "**留言墙**" 感受成员视角

### 深度阅读（2 小时）

按顺序阅读：

1. **三大宪制文件** → 理解联盟运作逻辑
2. **完整主文档** → 掌握世界观全貌
3. **典型案例** → 体会设定的现实张力

### 开发者/创作者

关注 **GPC 编号系统** 的技术细节：

- 2-3 字母俱乐部代号 ↔ 整数转换算法
- 层级编号：俱乐部(2-3 位) + 队(4 位) + 据点(3 位) + 研究员(5 位)
- 2063/2065 年升级路线图（GPC40 → GPC63 → GPC65）

## 🤝 参与贡献

本设定集采用 **开放世界观** 模式，欢迎：

- **补充设定**：完善某个俱乐部/队/据点的详细档案
- **技术实现**：用代码实现 GPC 编号解析器（已有 Haskell/Python/C++参考）
- **翻译工作**：将核心文档翻译为其他语言
- **衍生创作**：基于 GSCU 世界观的小说、漫画、游戏（需符合[亚文化指导方针](./GSCU介绍及附件文档/GSCU亚文化指导方针.md)）

### 贡献规范

1. 所有新增设定需符合现有**编号系统**和**历史时间线**
2. 重大设定变更需通过"一般性决议"流程（提交 Issue 讨论 2 周）
3. 保持"伪档案体"风格：客观、技术化
